# ATM Project (Beginner Level)

## 📌 Overview
This is a simple ATM Simulation Project built in Python. It is designed for beginners to understand basic programming concepts without using external modules.

This project demonstrates how an ATM system works using basic Python concepts like functions, loops, and conditional statements.

---

## ⚙️ Features
- Check account balance  
- Deposit money  
- Withdraw money  
- Simple menu-driven system  

---

## 🧠 Technologies Used
- Python  
- Basic Programming Concepts  
  - Functions  
  - If-Else Statements  
  - Loops (while loop)  
  - User Input Handling  

---

## 🚀 How to Run
1. Make sure Python is installed on your system  
2. Download or clone this repository  
3. Open terminal / command prompt in project folder  
4. Run the file using:# My-ATM-project-
"Basic Python ATM project for beginners implementing simple banking operations using functions and conditional logic
